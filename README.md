# MMA 360 Degrees Academy Website

## Files Included
1. `index.html` - Main HTML structure
2. `styles.css` - Complete CSS styling
3. `script.js` - JavaScript for interactivity

## Assets Folder Structure

Create an `assets` folder in the same directory as your HTML file with the following images:

```
assets/
├── logo.png                    (MMA 360 logo)
├── training1.jpg               (Hero section image 1)
├── training2.jpg               (Hero section image 2)
├── training3.jpg               (Hero section image 3)
├── training4.jpg               (Hero section image 4)
├── map-adyar.jpg              (Adyar location map)
├── map-thiruvanmiyur.jpg      (Thiruvanmiyur location map)
├── programs-hero.jpg          (Programs section hero background)
├── mma-training.jpg           (MMA Training program)
```

## How to Use

1. **Setup**:
   - Place all three files (index.html, styles.css, script.js) in the same folder
   - Create an `assets` folder in the same directory
   - Add all required images to the `assets` folder

2. **Image Requirements**:
   - Logo: 200x50px (approximately)
   - Hero training images: 400x300px (square format works best)
   - Location maps: 800x400px
   - Programs hero: 1920x800px
   - Program cards: 600x400px

3. **Customization**:
   - Colors: Main brand color is #FF5722 (orange). Change this in CSS if needed
   - Text: Update all content in index.html to match your needs
   - Contact info: Update phone numbers and addresses throughout the HTML
   - Social media: Add your WhatsApp number to the WhatsApp button link

4. **Features**:
   - Responsive design for mobile and desktop
   - Smooth scrolling navigation
   - Interactive FAQ accordion
   - Form validation for contact form
   - Animated counters for statistics
   - Testimonial slider (ready for multiple testimonials)
   - Scroll animations

5. **Testing**:
   - Open index.html in a web browser
   - Test all navigation links
   - Try the FAQ accordion
   - Submit the contact form (currently shows alert, can be connected to backend)
   - Test on mobile devices

## Sections Overview

1. **Navigation**: Fixed header with logo and menu
2. **Hero**: Main intro with image grid and features

6. **Testimonials**: Student review slider
7. **FAQ**: Expandable question/answer sections
